Linha do Tempo Textual

Abaixo estão alguns dos principais pontos da história da tecnologia de comunicação:

  1. Surgimento das redes: Interconexão inicial de computadores para troca de dados de forma descentralizada.
  2. ARPANET: Criada em 19691, foi a primeira rede operacional de computadores e a base da Internet atual.
  3. TCP/IP: Protocolo unificado adotado em 1983 que permitiu que redes diferentes conversassem entre si.
  4. DNS: Sistema criado para traduzir números de IP complexos em nomes fáceis de ler (como google.com).
  5. Criação da WWW: Em 1989, no laboratório do CERN, surge a proposta de organizar a informação em páginas interligadas.

Criação da World Wide Web

O cientista britânico Tim Berners-Lee é considerado o pai da Web. Ele desenvolveu os três pilares fundamentais da navegação:

HTML
Linguagem de marcação usada para estruturar os textos e elementos de uma página.
HTTP
Protocolo de comunicação usado para transferir os dados da página entre as máquinas.
Navegador e Servidor Web
O navegador (software do cliente) solicita e exibe a página, enquanto o servidor armazena e envia os arquivos solicitados.

Citação Famosa: "A Web é mais uma criação social do que técnica." – Tim Berners-Lee.

Evolução: Web 1.0 vs Aplicações Web Modernas

Na Web 1.0(estática), os usuários apenas consumiam conteúdo de páginas simples sem interação. Já nas Aplicações Modernas2 (Web interativa), o usuário cria conteúdo, interage em tempo real e utiliza sistemas complexos diretamente pelo navegador.
