O que é JavaScript?

O JavaScript foi criado com a motivação de tornar as páginas web interativas, permitindo que elementos visuais mudassem e reagissem às ações do usuário sem a necessidade de recarregar a página inteira.

A diferença de execução é que, no navegador, o JavaScript roda do lado do cliente para manipular a interface e a experiência do usuário. Já no servidor, o JavaScript roda nos bastidores processando dados, acessando bancos de dados e respondendo a requisições de rede.

O conceito de sandbox no navegador é uma barreira de segurança que isola a execução do código JavaScript, impedindo que ele acesse arquivos locais do computador do usuário ou danifique o sistema operacional.

Os objetos globais fornecem funcionalidades prontas: o 'document' permite acessar e modificar a estrutura do HTML, o 'console' serve para enviar mensagens de depuração no terminal do navegador, e o 'alert' exibe uma caixa de diálogo de aviso na tela do usuário.
